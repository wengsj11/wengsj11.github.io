ubuntu安装nodejs

发表于 2017-05-29 | 分类于 linux | 阅读次数

1. 官网下载tar.gz源代码文件，例如：node-v4.2.1.tar.gz

2. 使用FileZilla将文件上传至ubuntu服务器（若在ssh连接环境下）

3. 解压源码

$ tar zxvf node-v4.2.1.tar.gz

4. 编译前首先确保系统安装来Python,gcc,g++,如果没有则安装

$ sudo apt-get install python

$ sudo apt-get install build-essential

$ sudo apt-get install gcc

$ sudo apt-get install g++

5. 编译安装

$ cd node-v4.2.1

$ sudo ./configure --prefix=/usr/local/node/4.2.1

$ sudo make

$ sudo make install

6. 配置NODE_HOME，进入profile编辑环境变量

$ vim /etc/profile

设置nodejs环境变量，在文件末尾添加如下内容:

之后按Esc键，:wq保存并退出。（若文件显示为只读，输入:w !sudo tee %强制保存）

7. 编译/etc/profile 使配置生效

$ source /etc/profile

验证是否安装配置成功

$ node -v

$ npm -v

输出 v4.2.1 和 2.14.7 表示配置成功。

（npm模块安装路径/usr/local/node/0.10.24/lib/node_modules/）

在命令行输入：$ sudo vim /root/.bashrc 并在文件末尾加入一行source etc/profile命令，保存。这样就大功告成了。
（特别要注意配置好/etc/profile后，将配置加入/root/.bashrc中；否则，配置的环境变量不会在root账户下生效）
